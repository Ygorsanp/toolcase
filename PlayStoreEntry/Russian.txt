Root ToolCase

SHORT DESCRIPTION
*****************

A great collection of tools to get more out of your android device!

FULL DESCRIPTION
****************

<b>ROOT REQUIRED (это приложение не работает на устройствах без root)</b>

<b>DISCLAMER:</b>
The features of this app are very powerful, be careful to use them properly.
THE DEVELOPER OF THIS APP IS NOT RESPONSIBLE FOR DAMAGES TO YOUR DEVICE!

<b>Менеджер приложений:</b>
★ Очистка данных или кэша приложений
★ Заморозка и разморозка приложений
★ Удаление приложений

<b>Flash Wizard:</b>
★ Установка zip прошивок
★ create flash queue for several zip files
★ Установка образов ядер
★ Установка образов recovery

<b>Init.d Эмулятор:</b>
★ run scripts at startup
★ no native init.d support required

<b>Build.prop Editor:</b>
★ Редактирование вашей прошивки

<b>Advanced Reboot:</b>
★ Перезагрузка в recovery или fastboot

<b>Widgets:</b>
★ Task Cleaner
★ Advanced Reboot
